# FCIT Robotics Society Website

A static, responsive website for the **FCIT Robotics Society**, showcasing the society’s mission, projects, team, collaborations, and a contact form for new members.

---

### Project Structure

```
|-- index.html        # Main HTML file
|-- style.css         # Stylesheet for layout and design
|-- script.js         # JavaScript for smooth scrolling
|-- AZtVGPy2sMhPEoJ5pzeN9Q-AZtVGPy227hTRHXS7u9KFQ-removebg-preview.png  # Robot image
|-- README.md         # Project documentation
```

### Pages & Sections

### 1. Navigation Bar

* Fixed header with smooth scrolling
* Links to About, Projects, Team, and Contact sections

### 2. Hero Section

* Society name and slogan
* Call-to-action button
* Robot illustration

### 3. About Us

* Introduction to FCIT Robotics Society
* Community stats (members, projects, workshops)

### 4. Projects

* Obstacle Avoidance Robot
* Line Following Robot
* Robotic Arm

### 5. Team

* Core team members with roles

### 6. Collaboration

* Partnership with IEEE FCIT Student Branch

### 7. Contact / Join Us

* Simple contact form
* Name, email, and interest message fields

### 8. Footer

* Copyright notice
* Contact email link

## Technologies Used

* **HTML5** – Structure
* **CSS3** – Styling & layout

* **Font Awesome** – Icons
